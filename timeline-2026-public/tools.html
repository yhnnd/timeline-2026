link


A 
<link>
 tag will be translated to a 
<div class="link" onclick="openLink(event)">
 tag.


<link to="book-reader.html?src=../folder/filename.txt" >
 will be translated to


<div class="link" onclick="openLink(event)" to="book-reader.html?src=../folder/filename.txt" >


⚠️ 
Automatically Enabled.



img


A 
<img src="../folder/filename.jpg">
 will be recognized as a html 
img
 tag.

A Scalable Vector Graphic <svg> will be recognized as a html svg tag ONLY when it is written in single line. The opening tag <svg> and the closing tag </svg> MUST be in one single line.

⚠️ 
This function is disabled by default. Use 
@command("enable-image-recognition")
 to enable this function.



thumbnail row


A 
<div class="row">
 will be recognized as a html 
div
 tag.

A 
<div class="row">
 can wrap a list of 
<img class="thumbnail">
 and will display a group of images in one line.

If the image with class name 
thumbnail
 is too small, use 
thumbnail-2x
.

⚠️ 
Automatically Enabled.



border



<border>
</border>
 will be translated to


<div class="border">
</div>


⚠️ 
This function is disabled by default. Use 
@command("enable-border-recognition")
 to enable this function.



video



@video resources/35_1713060169.mp4


will be translated to


<video width="100%" controls="" style="width: 100%;">
  <source src="resources/35_1713060169.mp4" type="video/mp4">
</video>




YouTube Video


@video width="790" height="444" src="https://www.youtube.com/embed/VXGRZ6O38ec" frameborder="0" allowfullscreen


will be translated to


<iframe width="790" height="444" src="https://www.youtube.com/embed/VXGRZ6O38ec" frameborder="0" allowfullscreen=""></iframe>


⚠️ 
This function is disabled by default. Use 
@command("enable-video-recognition")
 to enable this function.



Map



@map {id: 'map_2', center: {lat: 39.06508132468046, lng: 117.10561037063599}, zoomLevel: 15}


will be translated to a Leaflet map.

⚠️ 
This function is disabled by default. You can enable it in Settings.
